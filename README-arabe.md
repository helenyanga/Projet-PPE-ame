# Analyse du mot "Âme" en arabe - Partie Sara

## Graphies étudiées
- **روح** (rūḥ) - 30 URLs
- **نفس** (nafs) - 30 URLs

## Fichiers importants
- `tableaux/ar.html` : Tableau récapitulatif unique pour les deux graphies
- `URLs/ar-rouh.txt` : Liste des URLs pour روح
- `URLs/ar-nafs.txt` : Liste des URLs pour نفس

## Scripts utilisés
- `programmes/script_arabe_unifie.sh` : Script principal de traitement
- `programmes/tokenisation.py` : Tokenisation arabe avec CAMeL Tools

## Résultats
- **Total URLs** : 61
- **URLs réussies** : 48 URLs 
- **Taux de réussite** :~78% 

## Notes techniques
Certaines URLs n'ont pas pu être analysées en raison de :
- Blocage du scraping par le site
- Contenu généré en JavaScript
- Pages inaccessibles ou supprimées

Ceci est normal et démontre le bon fonctionnement du script (gestion des erreurs).

## Rapport sur âme on arabe:
Sur les 61 URLs collectées (30 pour روح et 31 pour نفس), 48 ont pu être analysées avec succès (78%).
 Les 13 URLs restantes n'ont pas pu être traitées en raison de limitations techniques (blocage du scraping, contenu JavaScript, pages inaccessibles). 
 Ce taux de réussite de 78% est considéré comme satisfaisant dans le domaine du web scraping automatique.
